# 🌍 Legends Of Aether

Um jogo RPG 3D de mundo aberto criado com Three.js.

## 🎮 Recursos

- Mundo aberto
- Sistema de XP
- Boss Fight
- Cristais coletáveis
- HUD profissional
- Minimapa
- Água animada
- Ciclo de dia/noite
- Sistema de level

## 🚀 Tecnologias

- HTML
- CSS
- JavaScript
- Three.js

## ▶️ Como jogar

WASD = mover  
SHIFT = correr  
SPACE = pular  
CLICK = atacar
